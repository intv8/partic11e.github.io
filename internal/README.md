## Installation

To install, you simply need to re-export the library features with your
`deps.ts` file.

```ts
export {...} from  "https://denopkg.com/partic11e/internal/mod.ts";
```

and then import them from your `deps.ts` file into the files they are needed.

```ts
import {...} "../deps.ts";
```
